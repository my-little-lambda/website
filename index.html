My Little Lambda

A book on functional programming by Richard Torruellas

About the book

We will be writing functions, and a lot of them. Small, predictable functions. We will test them. We will see how we can make programs that are manly made up of functions. We will combine them to make new functions. We will have fun.

So, if you're checking out this book, you might already have an idea of what functional programming is. Either way, I think you will get something from this book. We will be gradually going through many of the "major" FP concepts. Going through these concepts should allow you to take these ideas to your current project, regardless of language.

We will be mostly using JavaScript throughout this book. Although JavaScript is not considered a pure functional language, it is actually quite capable of implementing most of the concepts. Later in the book we will go over what a pure functional language is, but for now, lets focus on the things that will bring value to you today.
